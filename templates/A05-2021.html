A05-2021: Security Misconfiguration

Security misconfiguration occurs when security settings are not implemented correctly, left at insecure defaults, or become outdated. It is one of the most common and easily exploitable vulnerabilities.

Examples

  • Exposed Admin Panels
  • Default Credentials
  • Misconfigured Security Headers
