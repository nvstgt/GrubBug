A10-2021: Server-Side Request Forgery (SSRF)

Server-Side Request Forgery (SSRF) occurs when an attacker can make the server perform unintended HTTP requests, potentially accessing or manipulating internal resources.

Examples

  • Accessing Internal Resources
  • Exfiltration of Sensitive Data
  • Using SSRF for Port Scanning
